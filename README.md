# Bird Migration Globe Visualization 🌍

An interactive, animated globe-based visualization of bird migration routes built with **React**, **MapLibre GL**, and **Vite**.

## ✨ Features

- Animated migration paths between start and end points
- Zoom-based visibility for points and lines
- Filters for species and migration months
- Responsive and accessible design
- Ready to deploy on **Vercel**

## 📦 Installation

```bash
npm install
npm run dev
```

## 🚀 Deploy to Vercel

1. Push the contents to GitHub.
2. Visit [vercel.com/import](https://vercel.com/import) and select the repo.
3. Set framework as **Vite** (auto-detected).
4. Done!

## 🧭 Tech Stack

- React 18
- MapLibre GL JS
- Vite
- CSS Modules
