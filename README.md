# Legendary Bird Migration Dashboard

- Next.js, Deck.gl, Mapbox, Python ML/statistics via FastAPI.
- No external GenAI or paid APIs: all analytics is local/statistical/ML.
- Deploy to Vercel, configure your Mapbox public API key as env `NEXT_PUBLIC_MAPBOX_TOKEN`.
- Cleaned migration data and route GeoJSON files required in `/public/data/`.
